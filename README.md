# NestCommerce API

A scalable REST API built with NestJS and PostgreSQL.

## Features

- JWT Authentication
- Role-Based Access Control (RBAC)
- Users, Products, Reviews modules
- PostgreSQL with TypeORM
- Modular NestJS architecture

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- TypeORM
- JWT

## Installation

```bash
git clone https://github.com/your-username/nestcommerce-api.git
cd nestcommerce-api
npm install
```

```bash
npm run start:dev
```

### Environment Variables

- Create a .env file based on .env.example.

```bash
Run the App
npm run start:dev
```
